## Description
Build the floating RAG Chatbot widget. The frontend must have a nice toggle button, chat bubbles with typing animations, and scroll-to-bottom effects. The backend is configured as a Python FastAPI application ready to run alongside or integrate OpenAI & vector search services.

## Acceptance Criteria

- [x] Frontend widget sends POST request to `/api/chat` router.
- [x] A local mock API response handler is active inside Next.js to simulate chatbot RAG behavior if the backend is offline.
- [x] Python backend code containing FastAPI setup, routers, services, and requirements is written in `/backend`.
